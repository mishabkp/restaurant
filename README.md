# 🍽️ Kerala Restaurant Navigator

A premium, full-stack web application designed to help users explore and discover the finest culinary experiences across Kerala. Built with a focus on modern UI/UX, real-time analytics, and intelligent recommendations.

## 🚀 Key Features

### 🧠 Smart Recommendation Engine
- **Content-Based Filtering:** Uses a weighted scoring algorithm (Cuisine, Category, Rating) to suggest the perfect match for the user's mood.
- **Vibe-Based Discovery:** A premium, glassmorphic interactive modal for discovering restaurants based on the "Vibe" (Romantic, Family, Friends, etc.).

### 🎬 Cinematic User Experience
- **View Transitions API:** Butter-smooth page transitions that mimic native mobile app behavior.
- **Glassmorphism Design:** A modern, high-end visual aesthetic with glowing elements and backdrop filters.
- **Interactive Kerala Map:** Integrated Leaflet.js maps for district-wise restaurant exploration.

### 📊 Visual Analytics & Trends
- **Trending Now Radar:** Simulates real-time popular spots across the state with live activity counters.
- **Price Variation Charts:** Compare price distributions for different cuisines across districts.

### 🛠️ Admin Dashboard
- **Content Management:** Full CRUD operations for restaurants, menus, and stories.
- **Status Control:** Live switching of restaurant open/close status and order tracking.

## 💻 Tech Stack

- **Frontend:** HTML5, Vanilla CSS (Modular), ES6+ JavaScript.
- **Libraries:** [Leaflet.js](https://leafletjs.com/) (Maps), [Lottie-Web](https://github.com/airbnb/lottie-web) (Animations).
- **Backend:** Node.js, Express.js.
- **Architecture:** Modular Monolith with environment-aware API switching.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mishabkp/restaurant.git
   ```

2. **Backend Setup:**
   ```bash
   cd server
   npm install
   npm start
   ```

3. **Frontend:**
   Simply open `index.html` in a browser or use a live server extension.
